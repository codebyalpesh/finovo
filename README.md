# Finovo

**Finovo** is a smart financial management platform designed to simplify money management. Track expenses, manage budgets, and make informed investment decisions with ease.

## 🌐 Demo
Check out the live demo here: [Finovo Demo](https://codebyalpesh.github.io/finovo/)

## 🚀 Features
- **Expense Tracking:** Easily log and categorize your expenses.
- **Budget Management:** Set monthly budgets and monitor your spending.
- **Investment Insights:** Track investments and get actionable insights.
- **Responsive Design:** Fully responsive UI for desktop and mobile.
- **Modern UI:** Built with clean, professional design principles.

## 🛠️ Built With
- HTML5, CSS3, JavaScript
- [Bootstrap 5](https://getbootstrap.com/)
- Custom Design System (Finovo Tokens & Components)
